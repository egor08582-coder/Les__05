# Werken met Git en GitHub

In deze handleiding leer je stap voor stap hoe je werkt met **Git** en **GitHub**.  
Je leert onder andere hoe je een repository aanmaakt, bestanden toevoegt en je werk online deelt.

![Git en GitHub overzicht](https://git-scm.com/images/logos/downloads/Git-Icon-1788C.png)

---

## Onderdelen

### Lokale repository
Een *lokale repository* is de map op je eigen computer waarin je werkt aan je project.  
Hier voer je Git-commando’s uit zoals `add`, `commit` en `push`.

### Online repository
Een *online repository* is jouw project op **GitHub**.  
Hier kun je jouw werk delen met anderen of samenwerken in een team.

---

### Acties die je moet uitvoeren

- Maak een nieuwe repository aan op GitHub  
- Vink de README.md-optie aan  
- Clone de repository naar je laptop  
- Open het project in Visual Studio Code  
- Voeg je eigen tekst toe aan de README  
- Commit je veranderingen  
- Push de commit naar GitHub  

---

### Extra informatie

1. **Git Add** – Voeg wijzigingen toe aan de staging area  
2. **Git Commit** – Sla je veranderingen lokaal op met een beschrijving  
3. **Git Push** – Upload je commit naar GitHub  
4. **Git Pull** – Haal de nieuwste versie van GitHub op naar je computer  
5. **Git Status** – Controleer de status van je bestanden
